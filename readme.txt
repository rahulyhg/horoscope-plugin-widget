== Description ==

Wordpress Daily Horoscope - This nifty widget lets you add a Daily Horoscope Sidebar to your wordpress blog or website.
The design of the widget is adjustable and can be simply tweaked through the wordpress setup page to blend in smoothly with your website.

Let us know your suggestions, comments and concerns by filling out the contact form in [Zodiac Signs](http://www.bzodiac.com).

== Installation ==

1. Upload `horoscope-plugin-widget` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Add the Horoscope Widget to your sidebar through the 'Widgets' menu in WordPress
4. Enable the widget and Optionally adjust options

== Frequently Asked Questions ==

= The widget does not appear on my website? =
Go to Dashboard -> Appearance -> Widgets and ensure the AWeber Mailing List widget has been added to the appropriate sidebar.

= I am getting "The widget is disabled" =
Go to Dashboard -> Appearance -> Widgets -> Open the "Horoscope Widget" settings and enable the widget.
